# IOPN Testnet Wallet Checker

A lightweight wallet analyzer for the IOPN testnet.

Features

- Native OPN balance checker
- Transaction counter
- ERC20 token detection
- NFT mint detection
- Testnet activity score

Built with ethers.js.
